1. Purpose and Scope

PCS Online Catalog is a web application used to browse live device inventory, filter by multiple criteria, build request carts, and submit inventory requests. It supports buyer and admin workflows and integrates with external inventory APIs for synchronization.

2. Functional Documentation

2.1 User Roles

  • Buyer: sign in, browse catalog, filter/search, open device details, add to requested items, submit request.
  • Admin: buyer capabilities plus user management, approvals, catalog utilities, weekly special controls, and external sync.

2.2 Key Flows

  • Authentication: login, create account, forgot password, approval states.
  • Catalog browsing: category home, category detail pages, server-side pagination.
  • Device details modal: carousel, availability, specs, request add flow.
  • Requested items modal: editable price/qty, totals, submit action.
  • Admin tools: sync, clear, seed synthetic, seed realistic, apply image mapping.

3. Technical Documentation

3.1 Architecture

  • Frontend: React 18 + Vite 5 (src/App.jsx)
  • Backend: Node.js HTTP server (backend/server.mjs)
  • Database: SQLite (schema)
  • API docs: OpenAPI

3.2 External Inventory Sync

Preferred integration uses OAuth2 client credentials + subscription key; legacy Boomi basic-auth fallback is retained for backward compatibility.
