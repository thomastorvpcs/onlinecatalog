1. Architecture

  • Frontend: React + Vite (src/App.jsx)
  • Backend: Node.js HTTP server (backend/server.mjs)
  • Database: SQLite (schema)

2. API Documentation

  • OpenAPI: docs/openapi.yaml
  • Inventory API: docs/inventory-api.md

3. Authentication and Authorization

  • Bearer access tokens + refresh tokens
  • Password hashing with scrypt
  • Role-based admin route protection

4. Inventory Integration

  • Preferred: OAuth2 client credentials + subscription key
  • Fallback: legacy Boomi basic-auth flow

5. Local Run

  • npm run api (backend)
  • npm run dev (frontend)
